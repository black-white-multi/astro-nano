## 安装NodeJS v20.17LST

https://nodejs.org/en/download/prebuilt-installer

## node自带npm

升级npm  
npm install npm -g  
  
## 安装astro

npm install astro  
IDE vscode  
安装astro插件
  
## 编译astro项目

npm install  
npm run dev  
npm run build  
注意：npm install 卡住不动, 切换至国内镜像源  
npm config set registry https://registry.npmmirror.com  

## 升级Astro

npm install astro@lastest
